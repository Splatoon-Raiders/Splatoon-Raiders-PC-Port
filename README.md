## Key Features
- **Treasure-hunting campaign** — explores the mysterious Spirhalite Islands
- **Ink-based combat** — fights Salmonids with Splatoon weapons and movement
- **Deep Cut story** — follows Shiver, Frye, Big Man, and the Mechanic
- **Gadget customization** — improves tanks, weapons, equipment, and the Hideout Ship
- **Three difficulty settings** — includes Tourist, Raider, and Survivalist
- **Multiplayer support** — allows two to four players online or through local wireless
- **Amiibo support** — unlocks outfits for the Mechanic with Splatoon amiibo
- **PC port tracking** — monitors Nintendo, Windows, and Steam platform announcements
- **TypeScript utility** — prints release status, platform data, countdown, and Nintendo pages

## Getting Started
1. **Open the Nintendo page** using the button above.
2. **Check the release date** and current software information.
3. **Review player modes** before planning online or local wireless sessions.
4. **Prepare about 20 GB** of storage for the estimated game size.
5. **Choose a difficulty** that fits your preferred combat experience.
6. **Run the TypeScript tool** for release status, platform data, or countdown information.

**Note:** A Nintendo Switch emulator is not the same as a native PC port. A Windows version requires a separate platform announcement.
